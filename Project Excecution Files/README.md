# Project Excecution Files

Archive directory containing backup copies of development files from earlier stages of the project. These are kept for reference and are not used by the live application.

## Contents

- `Flask/` - Backup of Flask application files (app1.py, templates, static)
- `Model/` - Backup of model development notebook

The primary working versions are in the sibling `Flask/` and `Model/` directories at the repo root.
