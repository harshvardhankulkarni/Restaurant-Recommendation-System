# Flask

Flask web application for the restaurant recommendation system.

## Files

- `app1.py` - Flask application entry point. Routes: `/` (home), `/recommend` (search form), `/result` (POST, returns recommendations), `/analytics` (rating distribution chart)
- `requirements.txt` - Python dependencies (Flask, pandas, scikit-learn, etc.)
- `Final_Development_Phase.ipynb` - Jupyter notebook used during model development
- `restaurant1.csv` - Dataset with 3,225 restaurants (name, rate, Mean Rating, cuisines, cost)
- `TODO.md` - Development notes and pending items
- `static/` - Static assets (images, background)
- `templates/` - Jinja2 HTML templates
